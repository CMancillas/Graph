Detailed Description

template<typename T>
class Graph< T >

Esta clase template ofrece una implementación de grafo (grafica) para agregar, eliminar elementos, y consultar el valor de cada nodo (vertice), entre otras.

Template Parameters
TTipo de elementos almacenados en el grafo.
